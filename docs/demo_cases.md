# Demo 案例

## 案例 1: 企业知识库问答

**场景**: 上传企业文档后，基于知识库进行问答

```
用户: 如何维护扫地机器人的HEPA滤网？

系统:
  → Query Router: knowledge_qa
  → RAG 检索: 找到 maintenance.txt 中相关内容
  → 证据过滤: 2条证据，最高分 0.85
  → 回答: "根据知识库资料，HEPA滤网需要每3个月更换一次，
           每周用软毛刷清洁表面灰尘。"
  → 置信度: high
  → 来源: maintenance.txt
```

**验证命令**:
```powershell
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"如何维护HEPA滤网？"}'
```

## 案例 3: 表格分析

**场景**: 上传电池测试CSV数据，进行统计分析

```
CSV内容:

3.7,1.5,25,100

3.5,1.3,35,99.0

输出:
## 表格分析结果
| 列名 | 最小值 | 最大值 | 均值 |
| 电压 | 3.5    | 3.7    | 3.6  |
| 温度 | 25     | 35     | 30.0 |
```
