# 动力电池知识库

## 1. 电池类型

### 1.1 三元锂电池 (NCM/NCA)
- 能量密度高（200-260 Wh/kg）
- 适合乘用车
- 热稳定性相对较差，需要完善的BMS管理

### 1.2 磷酸铁锂电池 (LFP)
- 安全性好，热稳定性高
- 循环寿命长（3000-5000次）
- 能量密度中等（140-160 Wh/kg）
- 适合商用车和储能

## 2. 充电策略

### 2.1 快充
- 快充功率通常为50-350kW
- 充电倍率1C-4C
- 需要在20%-80% SOC区间进行
- 温度需控制在15-45°C

### 2.2 低温充电风险
- 低温（<0°C）下快充可能导致锂枝晶生长
- 锂枝晶会刺穿隔膜，引发内部短路
- 低温快充会加速容量衰减，每100次低温快充循环可导致容量额外衰减3-5%
- 低温充电前应先预热电池至10°C以上
- 建议在-10°C以下禁止快充，-20°C以下禁止充电

### 2.3 高温充电限制
- 充电温度超过45°C会加速电解液分解
- 超过60°C可能触发热失控
- 建议充电温度区间为10-40°C

## 3. 电池管理系统 (BMS)

### 3.1 核心功能
- 电压监测：每颗电芯电压差需<50mV
- 温度监测：多点测温，精度±1°C
- SOC估算：误差<5%
- SOH估算：基于容量衰减和内阻增加
- 均衡管理：被动均衡（电阻放电）或主动均衡（能量转移）

### 3.2 保护机制
- 过充保护：单芯电压>4.25V触发
- 过放保护：单芯电压<2.8V触发
- 过温保护：电芯温度>65°C断开
- 短路保护：响应时间<100μs

## 4. 电池测试标准

### 4.1 国标测试项目
- GB/T 31484：循环寿命测试（500次循环容量保持率≥90%）
- GB/T 31485：安全性能测试（过充、过放、短路、挤压、针刺）
- GB/T 31486：电性能测试（容量、内阻、自放电率）

### 4.2 测试数据分析要点
- 容量衰减曲线：循环次数 vs 容量保持率
- 内阻增长趋势：循环次数 vs DC内阻
- 温度分布：不同工况下的电芯温度分布
- 电压一致性：电芯间电压标准差

## 5. 常见故障与对策

### 5.1 容量衰减过快
- 原因：高温存储、过充过放、低温快充
- 对策：优化热管理、调整充放电策略、定期均衡

### 5.2 电芯不一致
- 原因：制造公差、温度梯度、老化差异
- 对策：定期均衡、筛选配对、改善散热

### 5.3 热失控
- 原因：内部短路、过充、外部高温
- 对策：BMS多级保护、热隔离设计、消防系统

| SOC | State of Charge，荷电状态（%） |
| SOH | State of Health，健康状态（%） |
| DOD | Depth of Discharge，放电深度（%） |
| C-rate | 充放电倍率（1C=1小时充满/放完） |
| BMS | Battery Management System |
| 锂枝晶 | 低温充电时在负极表面形成的针状锂金属 |
